SpentSpentSpent privacy policy

SpentSpentSpent mobile application does not collect any data on the user. The user accounts are linked to the server the uesr is using (it's own our third party). The developers of SpentSpentSpent do not provide public servers (other than the demo)

SpentSpentSpent server itself does not collects the following data:

  • Email: for the sole purpose of logging into the application and sending recurring expense report.
  • Location: Only when the user allows the application to use the location when creating a new expense. This data is not used to create any profile

SpendSpentSpent developers do not own or share with anyone the data mentionned above the data collected is sent to the host of the server. Also the data is not processed for analytics purposes.
