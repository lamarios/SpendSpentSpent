SpendSpentSpent Privacy Policy

The SpendSpentSpent mobile application does not collect any personal data from users. User accounts are linked to the server they choose to use, whether it is a self‑hosted or third‑party server. The developers of SpendSpentSpent do not operate public servers except for an optional demo server.

When using a SpendSpentSpent server, the following information may be collected:

  • Email address:required solely for authentication, password recovery, and to send recurring expense reports.
  • Location data:captured only when the user explicitly permits the application to use location services during expense creation. This information is not used to build a user profile.

The SpendSpentSpent developers neither own nor share the data listed above. All collected data is transmitted exclusively to the host of the server and is never processed for analytics purposes.

Enabling the notification listener does not result in any data being sent to the server; all processing occurs locally on the device.
